Data Visualization

Libraries

D3.js – JavaScript based data visualization library

NVD3 – Re-usable charts for d3.js

Arbor.js – jQuery graph visualization library

Cubism.js – a d3.js plugin for visualizing time series

Dance – A simple data-driven visualization framework based on Data.js and Underscore.js

Envision – A Javascript library for creating fast, dynamic and interactive time series visualizations

Sigma.js – Javascript library to display interactively static and dynamic graphs

Flot – JQuery library for for line graphs and bar charts

Amcharts – JavaScript charting library

Highcharts – charting library written in pure HTML5/JavaScript, offering intuitive, interactive charts to your web site or web application

Raw – Raw is an open web app to create custom vector-based visualizations on top of the amazing D3.js library through a simple interface. – NOT INTERACTIVE

Vis.js – Javascript visualization library

dimple – An object-oriented API for analytics powered by d3. It aims to give a gentle learning curve and minimal code to achieve something productive

Chartist.js – Open Source JS Library For Responsive Charts

Dygraphs – dygraphs is a fast, flexible open source JavaScript charting library

Tools

Google Charts – A collection of simple to use, customizable and free to use interactive charts and data tools

Flourish – plug and play tool for making interactive data visualizations

GraphCommons – collaborative platform for mapping, analyzing and publishing data-networks

Voyant – open source text analysis tool
